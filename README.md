## Input and communication methods supported + hardware tested:  
### WinApi version:  
- Uses ctypes to call mouse inputs. No hardware needed.  

### Serial version:  
Tested with:
- Arduino Leonardo R3  

### Ethernet version:  
Tested with:  
- Arduino Leonardo R3 + Ethernet Shield + USB to Ethernet adapter (Optional)  

### Wi-Fi version:   
Tested with:  
- Raspberry Pi Pico W
